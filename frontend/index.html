TradeTally Free Trading Journal and Investment Tracker for Day Traders

TradeTally is a free open-source trading journal and investment tracker built for active traders. Track every trade, import broker history, review analytics, and improve execution with practical workflows for stocks, options, and long-term portfolio management.

Trading Journal Software with Broker Import

TradeTally gives you a complete trading journal workflow: log entries and exits, save notes and charts, tag setups, and review outcomes over time. You can auto-sync with Charles Schwab and Interactive Brokers or use CSV imports from Lightspeed, Webull, TradingView, TradeStation, Tradovate, Questrade, and other brokers.

Investment Tracker and Performance Analytics

As an investment tracker, TradeTally helps you monitor holdings, review unrealized and realized P&L, and analyze performance by symbol, strategy, and time period. Advanced metrics and AI-assisted insights make it easier to identify what is working and what is hurting your trading results.

Open Source and Self-Hosted Trading Journal

TradeTally is open source and can be self-hosted with Docker if you prefer full infrastructure control. Cloud users can get started immediately, while self-hosted users can run the same product with their own data policies, deployment process, and security controls.
